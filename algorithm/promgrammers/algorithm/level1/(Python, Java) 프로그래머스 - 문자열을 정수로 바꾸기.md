## Python 풀이
```python
def solution(s):
    return int(s)
```

## Java 풀이
```java
class Solution {
    public int solution(String s) {
        return Integer.parseInt(s);
    }
}
```

## Kotlin 풀이
```kotlin
class Solution {
    fun solution(s: String): Int {
        return s.toInt()
    }
}
```
